## 2. 画面詳細定義

### S01: ログイン画面

#### 画面概要
- システムへのログイン認証を行う

#### 画面項目

| 項目名 | 項目ID | 種類 | 必須 | 備考 |
|--------|--------|------|------|------|
| メールアドレス | email | テキスト入力 | ○ | ログインID |
| パスワード | password | パスワード入力 | ○ | マスク表示 |
| ログインボタン | btn_login | ボタン | - | クリックで認証処理 |

#### 画面遷移
- ログイン成功 → S02 (ダッシュボード)
- ログイン失敗 → エラーメッセージ表示

---

### S02: ダッシュボード

#### 画面概要
- ログイン後のトップ画面
- 本日の日報ステータス、未読コメント、今週の活動サマリーを表示

#### 画面項目

| セクション | 項目名 | 内容 |
|-----------|--------|------|
| ヘッダー | ユーザー名 | ログイン中のユーザー名表示 |
| ヘッダー | ログアウトボタン | ログアウト処理 |
| 本日の日報 | ステータス表示 | 未作成/作成中/提出済み |
| 本日の日報 | 日報作成ボタン | S04へ遷移（未作成時のみ表示） |
| 本日の日報 | 日報編集ボタン | S04へ遷移（作成中時のみ表示） |
| 通知エリア | 未読コメント件数 | 上長からのコメント通知 |
| 活動サマリー | 今週の訪問件数 | 当週の訪問記録総数 |
| 活動サマリー | 今月の訪問件数 | 当月の訪問記録総数 |
| クイックメニュー | 日報一覧リンク | S03へ遷移 |
| クイックメニュー | 顧客一覧リンク | S06へ遷移 |

#### 画面遷移
- 日報作成/編集ボタン → S04
- 日報一覧リンク → S03
- 顧客一覧リンク → S06

#### 営業担当者向け表示
- 自分の日報ステータス
- 自分への上長コメント通知

#### 上長向け表示
- 部下の日報提出状況（一覧）
- 未確認日報件数

---

### S03: 日報一覧画面

#### 画面概要
- 日報の一覧を表示し、検索・絞り込みが可能

#### 画面項目

**検索条件エリア**

| 項目名 | 項目ID | 種類 | 備考 |
|--------|--------|------|------|
| 期間（開始） | date_from | 日付入力 | デフォルト: 1週間前 |
| 期間（終了） | date_to | 日付入力 | デフォルト: 本日 |
| 営業担当者 | employee_id | ドロップダウン | 上長の場合: 全部下を選択可能<br>営業の場合: 自分のみ |
| ステータス | status | チェックボックス | 下書き/提出済み/確認済み |
| 検索ボタン | btn_search | ボタン | 検索実行 |

**一覧表示エリア**

| 列名 | 内容 | ソート | 備考 |
|------|------|--------|------|
| 日付 | 日報の日付 | ○ | yyyy/mm/dd形式 |
| 営業担当者 | 作成者名 | ○ | - |
| ステータス | 日報のステータス | ○ | バッジ表示 |
| 訪問件数 | その日の訪問記録数 | - | 件数表示 |
| コメント | コメント有無 | - | あり/なしアイコン |
| 操作 | アクションボタン | - | 詳細/編集ボタン |

#### 画面遷移
- 詳細ボタン → S05 (日報詳細画面)
- 編集ボタン → S04 (日報編集画面)
- 新規作成ボタン → S04 (日報作成画面)

#### 権限制御
- **営業担当者**: 自分の日報のみ表示・編集可能
- **上長**: 自分と部下の日報を表示可能、コメント追加可能

### S04: 日報作成・編集画面

#### 画面概要
- 日報の新規作成および編集を行う
- 訪問記録を複数行追加可能

#### 画面項目

**基本情報エリア**

| 項目名 | 項目ID | 種類 | 必須 | 備考 |
|--------|--------|------|------|------|
| 日付 | report_date | 日付表示 | - | 新規作成時: 本日（変更不可）<br>編集時: 対象日付表示 |
| 営業担当者 | employee_name | テキスト表示 | - | ログインユーザー名（変更不可） |

**訪問記録エリア**

| 項目名 | 項目ID | 種類 | 必須 | 備考 |
|--------|--------|------|------|------|
| 顧客名 | customer_id | ドロップダウン | ○ | 顧客マスタから選択 |
| 訪問時間 | visit_time | 時刻入力 | - | HH:MM形式 |
| 訪問内容 | visit_content | テキストエリア | ○ | 複数行入力可能 |
| 行追加ボタン | btn_add_visit | ボタン | - | 訪問記録行を追加 |
| 行削除ボタン | btn_delete_visit | ボタン | - | 該当行を削除 |

- 訪問記録は最低1行必須
- 並び替え可能（ドラッグ&ドロップまたは上下ボタン）

**課題・予定エリア**

| 項目名 | 項目ID | 種類 | 必須 | 備考 |
|--------|--------|------|------|------|
| Problem（今日の課題・相談） | problem | テキストエリア | ○ | 複数行入力可能 |
| Plan（明日の予定） | plan | テキストエリア | ○ | 複数行入力可能 |

**操作ボタンエリア**

| ボタン名 | 動作 | 備考 |
|----------|------|------|
| 下書き保存 | ステータス「下書き」で保存 | バリデーションなし |
| 提出 | ステータス「提出済み」で保存 | 必須項目チェックあり |
| キャンセル | 編集内容を破棄して一覧へ戻る | 確認ダイアログ表示 |

#### 入力チェック（提出時）
- 訪問記録: 最低1件必須
- 各訪問記録: 顧客名、訪問内容は必須
- Problem: 必須
- Plan: 必須

#### 画面遷移
- 保存成功 → S03 (日報一覧) または S05 (日報詳細)
- キャンセル → S03 (日報一覧)

| Plan | 明日の予定 | テキスト表示 |

| 項目 | 内容 |

| 項目名 | 項目ID | 種類 | 必須 |
|--------|--------|------|------|

#### 画面遷移
- 編集ボタン → S04 (日報編集画面)
- 戻るボタン → S03 (日報一覧画面)

- **営業担当者**: 自分の日報のみ閲覧・編集可能、コメント追加不可
- **上長**: 自分と部下の日報を閲覧可能、コメント追加可能

## 5. 補足事項

### ステータス遷移

日報のステータスは以下のように遷移:

```
下書き → 提出済み → 確認済み
   ↑         ↑
   └─────────┘
   (営業が再編集可能)
```

- **下書き**: 営業担当者が作成中
- **提出済み**: 営業担当者が提出、上長確認待ち
- **確認済み**: 上長が確認完了

### コメント種別

- **Problem用コメント**: Problemセクションへのコメント
- **Plan用コメント**: Planセクションへのコメント
- **全体コメント**: 日報全体へのコメント（オプション）

### 今後の拡張検討項目

1. 通知機能（メール・プッシュ通知）
2. 日報テンプレート機能
3. 添付ファイル機能（訪問時の写真など）
4. レポート・分析機能（訪問回数、顧客別訪問履歴など）
5. カレンダービュー
6. 一括インポート・エクスポート機能
